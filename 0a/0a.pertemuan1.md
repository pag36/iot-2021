# Pertemuan ke 1: Konsep dan Arsitektur IoT (Teori)

## Topik Bahasan
* Mahasiswa mampu memahami konsep dan arsitektur IoT
* Mahasiswa dapat merancang secara sederhana tentang arsitektur IoT kasus sederhana

- Edge  Computing
